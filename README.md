# Core Java Practice

## Overview
This repository contains Core Java practice programs created to strengthen fundamental concepts through hands-on coding.

## Topics Covered
- Basic syntax and control flow
- OOP concepts (classes, objects, inheritance, polymorphism)
- Exception handling
- Collections
- Practice programs and examples

## Purpose
This repository is meant for **learning and practice**, not production-ready code.
Programs are added incrementally as concepts are practiced.
